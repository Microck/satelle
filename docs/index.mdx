Satelle sends durable Computer Use work to an operator-controlled Host. Start
with the [first session tutorial](/docs/tutorial/first-session), then use the
how-to guides for day-to-day operation.

> Satelle is pre-release. Build it from source. The npm package names in this
> repository are not published installation paths yet.

- **Host administrator:** [Onboard a native Host](/docs/how-to/setup-host)
- **CLI user:** [Command reference](/docs/reference/commands)
- **Security reviewer:** [Security boundaries](/docs/explanation/security-boundaries)
